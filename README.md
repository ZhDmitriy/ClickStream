# ClickStream

Анализ воронки продаж (ClickStream) по переходам пользователей декстопного и мобильного приложения. 

Библиотеки: 
1. Pandas
2. Matplotlib

Любую воронку, как и эту можно детализировать все глубже и глубже, чтобы вытаскивать новые инсайты.
